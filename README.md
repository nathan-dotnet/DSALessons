# CSharpLesson

A simple .NET console application containing basic data structures and algorithms practice exercises.

## Lessons Included

- `LargestEvenNumber` - find the largest even number in an array.
- `CountEvenNumbers` - count the number of even numbers in an array.
- `LargestNumber` - find the largest number in an array.
- `SumArray` - sum all values in an array.
- `SensorReadings` - find the highest and lowest sensor readings.
- `SensorCount` - count readings per sensor ID.
- `FirstDuplicateSensor` - find the first duplicate sensor ID.
- `FirstDuplicateCharacter` - find the first duplicate character in a string.

## Run the Project

1. Open the project folder in Visual Studio or your preferred editor.
2. Update `Program.cs` to call the lesson you want to run.
3. Build and run the project.

## Example

In `Program.cs`, uncomment one of the lesson method calls:

```csharp
// LargestEvenNumber.Run();
// CountEvenNumbers.Run();
// LargestNumber.Run();
// SumArray.Run();
// SensorReadings.Run();
// SensorCount.Run();
// FirstDuplicateSensor.Run();
FirstDuplicateCharacter.Run();
```

Then execute the application.
